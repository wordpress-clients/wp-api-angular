  • getList(params, data, headers)

    Get page list

    Parameters
    ParamTypeDetails
    paramsobject

    Optional: {Object.} – Map of strings or objects which will be serialized with the paramSerializer and appended as GET parameters.

    dataobject

    Optional: {string|Object} – Data to be sent as the request message data.

    headersobject

    Optional: {Object} – Map of strings or functions which return strings representing HTTP headers to send to the server. If the return value of a function is null, the header will not be sent. Functions accept a config object as an argument.

    Returns
    Object

    Promise

    Example

    $wpApiPages.getList({
 page: 1,
 per_page: 10,
 "filter[orderby]": "date"
 "filter[orderby]": "asc"
});
